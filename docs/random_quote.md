## random quote

Generate a random quote

### Synopsis

Generate a random quote based on author name if supplied.

```
random quote [flags]
```

### Options

```
  -a, --author string   Full name of the author. To list Available authors run the following:
                        $random author --all
  -h, --help            help for quote
```
